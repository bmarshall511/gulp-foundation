    map-fetch

    Since 2.0.1
    @function map-fetch($map, $keys) { ... }

    Description

    Helper functions that returns the value of a scss array variable.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $map

    The scss array variable

    		Variable— none
    $keys

    Key of the array you want to return.

    		List— none

    Returns

    Mixed —

    Value of the sprcified scss array variable key.

    Example

    Basic Usage Sass

    $header-styles: (
  small: (
    'h1': ('font-size': 24),
    'h2': ('font-size': 20),
    'h3': ('font-size': 19),
    'h4': ('font-size': 18),
    'h5': ('font-size': 17),
    'h6': ('font-size': 16),
  ),
  medium: (
    'h1': ('font-size': 48),
    'h2': ('font-size': 40),
    'h3': ('font-size': 31),
    'h4': ('font-size': 25),
    'h5': ('font-size': 20),
    'h6': ('font-size': 16),
  ),
);

h1 {
   font-size: map-fetch($header-styles, small h1 font-size);
}

    CSS Output

    h1 {
  font-size: 3rem;
}

    • Ben Marshall

    font-face

    Since 2.0.1
    @mixin font-face($name, $path, $weight: null, $style: null, $exts: eot woff2 woff ttf svg) { ... }

    Description

    Helper mixin that generates font-face output.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $name

    The name of the font family

    		String— none
    $path

    Relative path from src to the font files.

    		String— none
    $weight

    Font weight.

    		Stringnull
    $style

    Font style.

    		Stringnull
    $exts

    List of avaiable file extensions.

    		Listeot woff2 woff ttf svg

    Example

    Basic Usage Sass

    @include font-face('Custom Font', '../fonts/CustomFont/CustomFont-Bold', 900, normal, eot woff ttf svg);

    CSS Output

    @font-face {
  font-display: auto;
  font-family: 'Custom Font';
  font-style: normal;
  font-weight: 900;
  src: url("../fonts/CustomFont/CustomFont-Bold.eot") format("eot"),
       url("../fonts/CustomFont/CustomFont-Bold.woff") format("woff"),
       url("../fonts/CustomFont/CustomFont-Bold.woff") format("ttf"),
       url("../fonts/CustomFont/CustomFont-Bold.woff") format("svg")
}

    Output

    @font-face rule definitions.
